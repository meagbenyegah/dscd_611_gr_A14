# Trained Models

This folder stores trained machine learning models.

## Contents
- Serialized model files (.pkl, .joblib, .h5)
- Model metadata and parameters
- Model versioning information

## Example files
- `random_forest_v1.pkl` - Random Forest classifier
- `logistic_regression_v1.pkl` - Logistic Regression model
- `model_metadata.json` - Model training parameters and dates
