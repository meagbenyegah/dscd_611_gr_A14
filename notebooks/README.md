# Notebooks

This folder contains Jupyter notebooks for exploratory data analysis and experimentation.

## Contents
- Data exploration and visualization notebooks
- Feature analysis notebooks
- Model experimentation notebooks
- Results analysis notebooks

## Naming convention
- `01_data_exploration.ipynb` - Initial data analysis
- `02_feature_engineering.ipynb` - Feature creation and selection
- `03_model_training.ipynb` - Model training experiments
- `04_model_evaluation.ipynb` - Model performance analysis
